# MuleGuard AI

AI/ML-based Suspicious Mule Account Detection system using XGBoost, risk scoring, SHAP explainability, and Streamlit dashboard.

## Features

- Suspicious mule account classification
- Risk score generation
- Critical account flagging
- SHAP-based explainability
- AI investigation console
- Downloadable prediction report

## Tech Stack

- Python
- XGBoost
- Scikit-learn
- Pandas
- Streamlit
- Plotly
- SHAP

## How to Run

```bash
pip install -r requirements.txt
python -m streamlit run app.py
